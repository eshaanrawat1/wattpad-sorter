# Wattpad Views Sorter

A Chrome extension that adds a "Sort by Views" control to Wattpad story lists by parsing read counts from story cards.

## Install (Unpacked)

1. Open Chrome and go to `chrome://extensions`.
2. Enable Developer mode.
3. Click "Load unpacked" and select this folder.
4. Open Wattpad and navigate to any page with story cards.

## Files

- `manifest.json` - Extension manifest (MV3)
- `content.js` - Content script (parsing, sorting, observer)
